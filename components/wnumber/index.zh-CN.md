## API

| 属性        | 说明                                       | 类型     | 默认值    |
| ---------- | ---------------------------------------- | ------ | ------ |
| bottomTitle| 底部标题                                  | String | ''     |
| rightTitle | 右边标题                                  | String | ''     |
| unit       | 单位                                     | String | ''     |
| numberTrend| 数值趋势, 填写raise或者drop自动处理数字    | "raise", "drop" | ''  |
