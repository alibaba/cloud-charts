## API

| 属性        | 说明                                   | 类型                      | 默认值    |
| --------- | ------------------------------------ | ----------------------- | ------ |
| className | 自定义样式名                               | String                  | ''     |
| height    | 高度                                   | String or Number        | 80     |
| status    | 状态，填写blue(正常)、orange(二级告警)、red(一级告警) | "blue", "orange", "red" | 'blue' |
