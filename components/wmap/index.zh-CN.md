## API

### 通用参数

| 成员      | 说明      | 类型      | 默认值        |
| ------- | ------- | ------- | ---------- |
| width   | 容器宽度    | Number  | 自适应父元素宽度   |
| height  | 容器高度    | Number  | 自适应父元素高度   |
| config  | 相关配置    | Object  | 见下方具体配置    |
| data    | 传入数据，必填 | Array   | 见下方具体配置    |
| geoData | 地理数据    | GeoJson | 根据config中的 |
