## API

| 属性        | 说明                                       | 类型     | 默认值    |
| ---------- | ---------------------------------------- | ------ | ------ |
| title   | 标题                                    | String | ''     |
| arrange | 排列方式                                | 'cross', 'normal' | 'normal' |
| height   | 高度                                          | String | '100px'     |


### Wcontainer.divider
