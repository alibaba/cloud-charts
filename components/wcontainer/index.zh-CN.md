## API

| 属性          | 说明                                 | 类型                | 默认值      |
| ----------- | ---------------------------------- | ----------------- | -------- |
| className   | 自定义样式名                             | String            | ''       |
| title       | 标题                                 | String            | ''       |
| arrange     | 排列方式                               | 'cross', 'normal' | 'normal' |
| height      | 高度                                 | String or Number  | '100%'   |
| operation   | 定义标题右侧操作                           | string or element | 无        |
| titleBorder | 是否展示标题底部的标线(border)，true显示、false隐藏 | boolean           | true     |


### Wcontainer.divider

分割线，不存在其他属性
